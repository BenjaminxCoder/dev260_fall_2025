## Projects

### Labs
- **Lab Loops and Conditionals:** Practiced control flow, branching logic, and basic C# program structure.
- **Lab Collections:** Worked with fundamental .NET collection types such as List<T> and Dictionary<T>.
- **Lab Stacks:** Implemented stack operations and understood LIFO behavior.
- **Lab Queues:** Explored queue operations and FIFO processing patterns.
- **Lab Hashsets:** Learned how hashing enables fast lookups and unique collections.
- **Lab Trees:** Built and traversed tree structures, practicing depth-first and breadth-first operations.

### Assignments
- **Assignment DS Foundations:** Introduced big-O notation, indexing trade-offs, and algorithm selection.
- **Assignment Linked Lists:** Implemented singly and doubly linked lists with insertion, deletion, and traversal.
- **Assignment Stacks:** Built a custom stack class and used stack-based algorithms.
- **Assignment Queues:** Designed queue structures and implemented circular queue logic.
- **Assignment Hashsets:** Explored collision handling, hashing strategies, and set operations.
- **Assignment Trees:** Created binary trees, performed traversals, and analyzed subtree operations.
- **Assignment Graphs:** Implemented graphs, BFS, DFS, and shortest-path algorithms.
